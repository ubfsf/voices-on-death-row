   (Or when deployed, add `/studio` to your domain)

## 📝 How to Add Content

### Adding a Voice Profile

```
Title: "Voices" → Click "Create"

Fill in:
- Person Name: "Ivan Kilgore"
- Photo: Upload a photo
- Biography: "Write their background..."
- Story: "What's their story?"
- Sentence Status: "On death row since..."

Save → Publish
```

**That's it!** It automatically appears on the Voices page.

---

### Adding a Letter

```
Title: "Letters" → Click "Create"

Fill in:
- Title: "Letter from California"
- Author: "John Smith"
- Content: Paste the full letter text
- Date Written: 2024-01-15

Save → Publish
```

**Done!** Letter appears in the archive.

---

### Adding Artwork

```
Title: "Art From Inside" → Click "Create"

Fill in:
- Title: "Haiti Drawing"
- Artist: "Anonymous - Death Row Artist"
- Image: Upload the artwork (shows in FULL COLOR)
- Description: Context about the piece

Save → Publish
```

**Beautiful!** Art displays in color on your Art page.

---

### Adding Podcast Episodes

```
Title: "Podcasts" → Click "Create"

Fill in:
- Title: "Episode 1: Voices from Texas"
- Guest: "Person's Name"
- Audio URL: "https://link-to-mp3-file.com/episode1.mp3"
- Description: "What listeners will hear..."

Save → Publish
```

**Instant!** Episode appears with player on Podcast page.
