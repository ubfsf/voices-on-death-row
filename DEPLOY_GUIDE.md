# Deploy to Live Website - Simple Steps

## Your Code is Ready! ✅

All your changes are **saved locally** but need to go to **GitHub** so Vercel can deploy them.

## The Easiest Way: GitHub Desktop

**No terminal needed! Just drag & drop.**

### Steps:

1. Download GitHub Desktop: https://desktop.github.com/
2. Open it and sign in with your GitHub account
3. Click "Clone a Repository"
4. Search for `ubfsf/voices-on-death-row`
5. Choose where to save it
6. GitHub Desktop clones it to your computer
7. Open both folders (current + cloned) in Finder
8. Copy these folders from current to cloned:
   - `src/`
   - `messages/`
   - `public/`
9. GitHub Desktop will show all the changes
10. Write a message like "Update descriptions and deploy"
11. Click "Commit to main"
12. Click "Push origin"
13. **Done!** Vercel deploys automatically

## What Happens Next

- ⏳ Vercel detects the push (1-2 minutes)
- 🔨 Vercel builds your site (1-2 minutes)  
- 🚀 Your live site updates (2-3 minutes total)

## Check Your Deployment

Go to: https://vercel.com/ubfsf/voices-on-death-row

Click "Deployments" to see the status.

## Your Live URL

Once deployed, you can access your site at the URL shown in Vercel dashboard.
